### log

功能：对基础50音的顺序以及打乱背记，加上浊音、拗音心记

初期策划：
·可以选择基础50音、浊音、拗音、长音（未做，还没学习到）的其中一类或多类型进行

·可以顺序或乱序

·选择出卡片正面展示的平假或者片假，点击卡片会显示对应的平假或片假

·请使用Chrome浏览器进行

后期规划:

·录入音频,可进行听写模式
